Creación de Documentos Científicos

Al igual que con Microsoft Word, con LaTeX es posible crear cualquier tipo de documento, sin embargo su enfoque va orientado a la creación de documentación técnica como Reportes científicos, Trabajos de Tesis, Artículos, etc...

% Inicio de nuestro documento.
\documentclass[12pt,letterpaper,oneside]{book}

% Uso de caracteres especiales
\usepackage[utf8]{inputenc}

% Especificamos metadatos como el título y el autor del documento
\title{Notas sobre \LaTeX}
\author{Emilio-Ernesto Hernández-Huerfano}

\begin{document}

% Portada básica
\maketitle
% Primer capítulo
\chapter{Veamos si funciona \LaTeX}
\section{Probando el procesamiento del texto}
\noindent El objetivo de este documento es verificar que el procesador de documentos lo está tomando de manera correcta. Es importante observar si los caracteres especiales de nuestro idioma son procesados correctamente: á é í ó ú niño güero.

\end{document}

La estructura de un documento está dividida en dos grupos


  • El grupo de instrucciónes que conforman el preámbulo del documento, es decir, aquellas instrucciones que son parte de la configuración: Tipo de documento, tipo de hoja, márgenes, colores, cabeceras y paquetes necesarios, entre otros. Estas se encuentran antes de la instrucción \begin{document}.
  • El grupo de instrucciones que detallan el cuerpo del documento o contenido textual y formato: Títulos, párrafos, imágenes, tablas, fórmulas, etc. Estas se encuentran entre las instrucciones \begin{document} y \end{document}.

En LaTeX, las instrucciones para el procesador de texto siempre inician con una diagonal invertida o backslash "\", para el compilador significa que a partir de ese caracter, viene una palabra que describe una acción. Existen 3 tipos principales de instrucciones:

  • Comandos sin argumentos, como el caso de \LaTeX.
  • Comandos con argumentos, como el caso de \chapter{...} o \documentclass[...]{book}.
  • Comandos conocidos como "environments", que tienen una instrucción de apertura y de cierre, como \begin{document} y \end{document}.

Incluyendo imágenes

Para poder añadir imágenes al documento, es necesario que nuestro primer paso sea agregar el paquete graphicx (\usepackage{graphicx}) a nuestro preámbulo, ya que este contiene las instrucciones necesarias para el manejo de gráficos.

Una vez añadido, podemos insertar los comandos necesarios para que latex inserte una imagen por nosotros, a continuación un ejemplo:

Incluyendo imágenes

\begin{figure}[h]
\centering
\includegraphics[scale=.06]{img/milla.jpg}
\caption{Mon amour :3}
\end{figure}

Como se puede notar, utilizamos un comando de tipo environment \begin{figure} ... \end{figure} para especificar que dentro de esas instrucciones de apertura y cierre estamos planteando los comandos necesarios para que nuestra imagen se despliegue correctamente.

El comando \centering solo indica que queremos que la imagen esté centrada. Si no especificamos dicho comando, nuestra imagen quedará orientada a la izquierda por default.

La instrucción \includegraphics{img/milla.jpg} está diciendo al procesador de texto el directorio en que la imagen que queremos incluir se encuentra. La configuración extra [scale=t] que notamos en la instrucción es para mencionar cuanto queremos que escale la imagen al incluirla en el documento, en donde t = 1 representa el tamaño original de la imagen, t < 1 quiere decir que la imagen será menor a la del tamaño original y t > 0 especifica que queremos ampliarla.

El comando \caption{...} especifica el texto que contendrá el pie de nuestra imagen. La numeración que se antepone de manera automática es generada de acuerdo al conteo de imágenes por capítulo.
